h1. Marketplace

This application is an excercise app. You can see it live on http://tboxmarketplace.herokuapp.com

h2. Ruby on Rails

This application requires:

* Ruby version 2.0.0
* Rails version 4.0.0


h2. Database

This application uses SQLite with ActiveRecord on development and Postgres in production

h2. Development

* Template Engine: ERB
* Testing Framework: RSpec and Factory Girl
